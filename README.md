# SIGF - Sistema Integrado de Gestión Fronteriza (Paso Pehuenche)

## 📌 Descripción del Proyecto
Prototipo funcional de interfaz web para la modernización del control aduanero en el Paso Fronterizo Pehuenche. Este sistema tiene como objetivo centralizar y agilizar los procesos de fiscalización, reduciendo los tiempos de espera y automatizando la validación de antecedentes mediante la interoperabilidad con instituciones externas.

## 🚀 Características Principales (Requerimientos Funcionales)
El prototipo abarca los siguientes módulos operativos:
* **Dashboard Operativo:** Panel de control con estadísticas en tiempo real y gráficos de flujo fronterizo de pasajeros y vehículos.
* **Control de Menores (PDI):** Módulo de validación de autorizaciones notariales para la salida del país de menores de edad.
* **Registro de Vehículos:** Control de tránsito internacional, registro de patentes y generación de documentos.
* **Declaración Fitosanitaria (SAG):** Interfaz para el control y registro de artículos declarados (animales, alimentos).
* **Módulo de Reportes:** Sistema de filtrado y simulación de exportación de datos en formatos PDF y Excel.

## 🛠️ Tecnologías Utilizadas
El proyecto es una aplicación frontend estática, diseñada bajo los principios de modularidad y usabilidad:
* **HTML5:** Estructuración semántica de las 8 vistas del sistema.
* **CSS3:** Estilos personalizados, implementación de *Dark Mode*, *Glassmorphism* y diseño 100% *Responsive*.
* **JavaScript (Vanilla):** Lógica de interfaz, navegación y validaciones visuales.
* **Chart.js:** Implementación de gráficos analíticos dinámicos.

## ⚙️ Instalación y Uso
Al ser un prototipo de interfaz sin backend integrado, no requiere instalación de dependencias, servidores locales ni bases de datos.
